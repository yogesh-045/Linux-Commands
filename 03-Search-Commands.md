# 1. find	
Searches for files and directories based on specified criteria.

# 2. grep	
Searches for a specific pattern or text within files.

# 3. locate	
Quickly finds files using a pre-built database.

# 4. which	
Displays the path of an executable command.

# 5. whereis	
Displays the binary, source, and manual page locations of a command.

# 6. type	
Identifies how a command is interpreted by the shell (alias, builtin, function, or executable).
