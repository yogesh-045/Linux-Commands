# whoami	
Displays the current logged-in user.

# who	
Displays information about users currently logged into the system.

# id	
Displays the User ID (UID), Group ID (GID), and group memberships.

# passwd	
Changes the password for a user account.

# su	
Switches from the current user to another user account.

# sudo	
Executes a command with administrative (root) privileges.

# groups	
Displays the groups to which a user belongs.

# users	
Displays the usernames of users currently logged into the system.
