# 1. ip a
Displays all network interfaces along with their IPv4 and IPv6 addresses.


# 2. ip link
Displays network interfaces and their operational status (UP/DOWN).


# 3. ip route
Displays the system's routing table and the default gateway.


# 4. hostname
Displays or changes the hostname of the Linux system.


# 5. ping
Tests network connectivity by sending ICMP Echo Request packets to a remote host.


# 6. ifconfig 
Displays or configures network interface settings.

# 7. netstat
Display network connections and ports.

# 8. wget
Download files from the Internet.

# 9. dig
Query DNS records

# 10. nslookup
DNS lookup

# 11. traceroute
shows the  packets take to a destination.

#  Interview Questions
# Q1. What is the difference between ip a and ifconfig?

ip a is the modern command from the iproute2 package and is recommended on current Linux systems.
ifconfig is an older (legacy) command from the net-tools package.
# Q2. What does the ping command do?

It tests network connectivity by sending ICMP Echo Request packets and measuring whether replies are received.

# Q3. What is the purpose of the hostname command?
It displays or changes the hostname of the Linux system.
